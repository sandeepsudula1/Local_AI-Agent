## 🎯 Path Resolution (Critical Design)

The app detects if running as `.exe` and configures paths accordingly:

```python
# In configs/settings.py

if getattr(sys, "frozen", False):
    # Running as PyInstaller .exe
    PROJECT_ROOT = Path(sys.executable).parent  # {app}\
    DATA_DIR = %APPDATA%\LocalAIAgent\data      # User-writable
else:
    # Running from source (dev)
    PROJECT_ROOT = os.path.dirname(__file__)
    DATA_DIR = PROJECT_ROOT / "data"
```

**Result:** Same code works in dev and production! ✅

## ⚠️ Known Limitations & Workarounds

| Issue | Impact | Workaround |
|-------|--------|-----------|
| Tesseract hardcoded path | OCR won't work if installed elsewhere | Set `TESSERACT_CMD` in `.env` |
| Windows Defender flags binary | User sees "Unknown Publisher" | None (normal for PyInstaller binaries) |
| First-run 300 MB download | Slow first startup (requires internet) | Pre-cache models in Pro version |
| Ollama required for all features | Must be running locally | Documented clearly for users |
| No bundled embeddings | Downloaded at runtime | Cached after first run |
